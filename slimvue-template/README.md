# slimvue使用说明

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and watch for development with slimapp
```
npm run watch
```
### Compiles and minifies for production
```
npm run release
```

### Lints and fixes files
```
npm run lint
```
